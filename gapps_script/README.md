# Using Clasp to manage Google Apps Script

## Setup

1. Install clasp

    ```bash
    npm install -g @google/clasp
    ```

2. Login

    ```bash
    clasp login
    ```

3. Clone a new project

    ```bash
    clasp clone "1s55R9e9Qb6pLRTqyRH4oaDmMeM2mDYsRIfBmZk1gxq0HhS3Y4dJeDtmf" --rootDir ./
    ```
4. Push changes

    ```bash
    clasp push
    ```
5. Deploy

    ```bash
    clasp deploy
    ```
